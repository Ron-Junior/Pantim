## Project Overview
**Pantim** é um clone de Fictionary/Jackbox - um jogo de diversão, humor e rapidez em rede local (LAN).

### Architecture
- **Backend**: Node.js + TypeScript + Socket.io (Entry: `./server/src/index.ts`)
- **Frontend**: SolidJS + Tailwind CSS (Entry: `./client/src/App.tsx`)
- **Build System**: Vite

### User Profiles
1. **Host** - Tela da TV (controle principal do jogo)
2. **Player** - Celular/Dispositivo dos jogadores

## Development Guidelines

### Code Patterns
- ✅ Use TypeScript para todo o código
- ✅ Frontend: `createStore` para estados de jogo complexos em SolidJS
- ✅ Use componentes funcionais com reatividade granular do Solid
- ✅ Mantenha lógica de Sockets separada da lógica de UI
- ✅ Compartilhe tipos entre server e client via pasta `/shared`

### Network Considerations
- ⚠️ Projeto roda em LAN - use IPs locais, **não localhost**
- ⚠️ Considere latência de rede local
- ⚠️ Implemente sincronização robusta entre Host e Players via Socket.io
